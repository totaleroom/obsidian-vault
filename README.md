# 🧠 Yura's Obsidian Vault

Personal knowledge base — owned, organized, and accessible.

**Managed with:** Hermes Agent (automated notes, captures, processing)
**Location:** `/home/hermes/obsidian-vault/`
**Access:** Local machine + Hermes VPS

## 📁 Folder Structure

| Folder | Purpose |
|--------|---------|

| `people/` | Contacts, relationships |
| `topics/` | Deep dives on topics |
| `notes/` | Atomic notes, ideas |
| `_templates/` | Reusable note templates |
| `_scripts/` | Templater/JS automation scripts |

## 🚀 Quick Start

### On Your Machine
1. Install Obsidian from [obsidian.md](https://obsidian.md)
2. Open vault → select `/home/hermes/obsidian-vault/` (via sync)
3. Enable plugins: Templater, Dataview, Banners

### Syncing Options
- **Obsidian Sync** (official, $4/mo)
- **iCloud** (macOS)
- **Dropbox**
- **Git** (via Obsidian Git plugin)
