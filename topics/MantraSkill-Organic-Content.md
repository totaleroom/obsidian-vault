## Strategi: "Organic Drop"

### Core Concept
Personal accounts (bukan brand) drop references ke MantraSkill dalam konteks nyata:
- Comment di thread/complain orang
- Story/post tentang pengalaman apply kerja
- Group posts dengan rekomendasi personal
- Reply chains saat ada yang bertanya

### Kenapa Ini Jalan
- **Trust**: Orang lebih percaya "temen yang udah pake" dari pada brand yang bilang "our product is great"
- **Algorithmic**: Engagement-based — comment, reply, share lebih high-value dari pada branded content
- **Tidak keliatan promo**: Tidak ada "IKLAN", "PROMO", "BUY NOW"
- **Scalable**: Batch content, stagger posting, vary account

### Perbedaan dengan Content Marketing Biasa

| Normal (Brand Voice) | Organic Drop (Personal Voice) |
|---------------------|----------------------------|
| "MantraSkill adalah sistem CV ATS..." | "Gue pake ini, beneran work" |
| "Tersedia 850+ alumni..." | "Temen gue udah pake, dapet kerja" |
| "Gratis 7 hari..." | "Btw ini gratis first, worth it" |
| Screenshot landing page | Screenshot hasil/skor ATS |

---

## Aturan Golden

1. **Tidak pernah brand voice** — no "MantraSkill presents..."
2. **Casual language only** — "beneran", "emang", "btw", "gue", "gw"
3. **Personal pronoun** — selalu "gue/gue/gw", tidak pernah third-person testimonial
4. **Timing stagger** — jangan posting sekaligus, spread berhari-hari
5. **Account variation** — kalau multi-account, vary writing style sedikit
6. **Real engagement** — reply comment, answer questions, jadi "orang" bukan bot
7. **No hashtag spam** — tidak ada #lowongankerja #kerjacepat
8. **Screenshot bukan logo** — kalau ada screenshot, show hasil/skor, bukan logo MantraSkill

## Platform Approach

### Facebook

**User type:** Indonesian job seekers, fresh grads, career switchers. Banyak group (lowongan kerja, career advice, industry-specific).

**Content format:**
- Story-style paragraph posts (longer, more narrative)
- Comment drops in groups
- Group recommendations
- Story replies to people asking for advice

**Best groups to target:**
- Lowongan Kerja Indonesia / Jakarta / Surabaya dll
- Industry-specific (Tech Jobs ID, Marketing Indonesia, Finance Career)
- Fresh grad / career advice groups
- Community groups (alumni specific)

**Tone:** Story-telling, more formal than Threads but still casual. Mix Indo + English.

---

### Threads

**User type:** Younger, more casual, professionals + career-curious. Public, shareable, scroll-heavy.

**Content format:**
- Short punchy posts (500 chars)
- Reply to complaint threads
- Quote posts
- Casual mentions

**Tone:** Very casual. Abbreviations okay. DMs-to-group-chat energy. Mix Indo-English slang.

## API Reference — Zernio

**Base URL:** `https://api.zernio.com/v1`
**Auth:** `Authorization: Bearer <API_KEY>`

### Create + Publish IMMEDIATELY (publishNow)

```bash
curl -X POST https://api.zernio.com/v1/posts \
  -H "Authorization: Bearer <KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Post text...",
    "publishNow": true,
    "visibility": "public",
    "platforms": [
      {
        "platform": "threads",
        "accountId": "<ACCOUNT_ID>"
      }
    ]
  }'
```

**CRITICAL:** `platforms` is an ARRAY of objects with `platform` + `accountId`. NOT `accountIds` at top level.
