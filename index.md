# Welcome to Your Second Brain

This vault is the shared knowledge layer between **Yura** and **Hermes Agent**.

---

## What Goes Where

### Inbox (🟡 Process Weekly)
Raw captures — ideas, links, half-formed thoughts. Review and move to proper folders.

### Daily Notes
One note per day. Habits, tasks, reflections. Access via calendar plugin.

### Projects
Anything with a goal, timeline, or deliverable. Use the `project` template.

### People
Contacts, conversations, context. Use the `person` template.

### Topics
Deep dives. Research, learning, summaries. Auto-linked via tags.

### Notes
Atomic, standalone ideas. Small, linkable, timeless.

---

## Tags in Use

```
#daily      — daily note
#project    — active project
#person     — contact
#topic      — deep dive
#idea       — unvalidated idea
#resource   — reference material
#question   — open question
#insight    — learned something
```

---

## Working with Hermes

Hermes can:
- Create notes in any folder
- Search and retrieve via `obsidian` skill
- Update notes from session summaries
- Process inbox items into proper notes

To ask Hermes to capture something:
> "Save this to Obsidian: [content]"
